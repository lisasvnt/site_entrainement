- Qu’est-ce qu’un langage de programmation ?

Un langage de programmation ou langage informatique est un mode de communication compréhensible par les machines. Qui nous permet de leur faire exécuter des programmes qui contiennent des listes d’instructions pour qu’elle réalise des tâches bien précise (calcules, logiciels, sites web, automatisation…).

Le seul langage vraiment compréhensible par les ordinateurs : Ce langage est appelé langage machine ou binaire est n’est composé que de deux caractères le 1 et le 0. De manière très simplifiée un ordinateur est un assemblage de circuit qui ne savent faire que deux choses. Laisser passer le courant (1) ou ne pas le laisser passer (0). Cette multitude de circuits s’ouvrant et se ferment à la suite permettent de rédiger des messages complexes.

  • A = 01000001
  • 1 = 00110001
  • Bonjour = 01000010 01101111 01101110 01101010 01101111 01110101 01110010
Traducteur binaire

Le langage binaire étant extrêmement dure à maîtriser que quasiment personne ne programme comme ça directement. On utilise des langages dit de haut niveau (le binaire est un langage de bas niveau). Ces langages de haut niveau traduisent le binaire est facilite grandement le travail des développeurs.

Présentation des grands langages de programmation :

- HTML :

HTML (Hyper Text Markup Language) : Il va permettre de créer et structurer des pages web. Il va former le squelette des sites et applications présentes sur internet. Ajouter des textes, des images, des liens, des titres, des paragraphes… Actuellement ce langage en est à sa cinquième version d’où son nom complet HTML5.

Site sur l'HTML

- JS :

JS (JavaScript) : Il va permettre d’animer les pages web en leur appliquant des fonctions. Comme par exemple en animant des boutons, ou des pop-ups. Sans ce langage les utilisateurs ne peuvent pas interagir avec la page.

Site sur le JS

- CSS :

CSS (Cascading Style Sheet) : Ces un langage qui permet d’appliquer aux pages web des styles tels que de la couleur, des tailles, des polices de textes, la disposition des éléments, ect… Son rôle est donc d’embellir les contenus web.

Site sur le CSS

Ces trois langages ne représentent qu’une petite partie des quelques 700 langages existants. On peut par exemple citer l’SQL pour la manipulation de données dans de large base. Le R pour les traitements statistiques. Le Matlab pour les mathématiques. Le C ou le C++ qui est très utilisé pour la création de logiciels ou de jeux-vidéo. Le PHP pour la programmation web côté serveur. Ou encore le python qui est un langage très polyvalent utilisant en programmation web, logiciels, statistiques ou encore en cyber sécurité…
